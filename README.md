## Tasks

### Build

本番ビルドの作成

```shell
bunx --bun astro build
```

### Dev

開発サーバーの起動

```shell
bunx --bun astro dev
```

### Lint

コードの静的解析

Inputs: FILES

```shell
bunx --bun eslint --fix $FILES
```

### Format

コードのフォーマット

Inputs: FILES

```shell
bunx --bun prettier --check --write $FILES
```

### Shell

devboxを有効化

```shell
devbox shell
```
